Globe with Circular Paths

  • The Internet started in 1950s
    • J.C.R. Licklider - psychologist from MIT (1950s)
      • Wanted better computer communication, thus created the framework for a network
  • Hot Potato Routing
    • Separate data small enough to be processes on the fly
      • Eventually lead to the idea of network packets
  • 1962: theory of packet switches finalized

Map of ARPANET

  • Eisenhower created ARPA
    • Originally for space tech but included CS
  • 1969: prototype created for packet switching
  • Taylor of ARPA given $1mil to create computer network (ARPANet)
    • Roberts created the network design (star mesh and mini pc as switches[IMPs])
    • Jan 1 1969: started project with BBN
    • Lots of problems
      • Time to live for packets
      • Where any errors occurred
      • How to communicate with IMP
    • End of 1969: project done
  • April 1971: 18 mainframes connected through ARPANet

WWW

  • Raymond Tomlinson: created first app - email
  • 1973: TCP/IP protocol created
  • June 9, 1992: Congress made ARPANet public
    • 50 web pages existed
  • Tim-Berners Lee: created WWW
  • 1993: Mosaic -> NetScape - basically a web browser
    • Increased Internet use by 341000%
