Navigator Homescreen

  • 1993: Mosaic created as web browser
  • Jim Clark co founded NetScape
  • Microsoft gained dominance
  • NetScape ignored Microsoft talks due to gaining popularity of NetScape
    • Oct 13, 1994: Navigator by NetScape released (web browser)
      • Very popular with +1mil downloads
  • Bill Gates sent a message to beat NetScape
    • Some say this was the start of the browser war
  • Microsoft and NetScape meeting: June 1995
    • This meeting was secretive and undocumented
    • MS - talked about how to work together
    • NS - received offer to buy firm $1m or they will steal
      • If true, MS broke antitrust
  • NetScape went public with no history and profits
    • End of first day: Clark turned $5m to $663m

Internet Explorer Logo

  • 1995: MS released Internet Explorer
    • The overall idea was to make IE better than NetScape version by version
    • MS sales team pushed hard to only install IE
      • Example: Compaq lose Windows license if it installed NetScape
  • NetScape charged money for browser
  • MS release IE for free
    • By IE 4.0: MS had 80% users on Internet
  • Sept 1997: browser war ends with MS victory
  • 1998: AOL bought NetScape
  • 1998: MS hit with antitrust suit
    • Gates seemed untruthful
    • MS was guilty according to judge
    • Appeals court rescinded order to split company
    • Gates retired and Steve Ballmer became CEO
    • Gates focused on philanthropy
